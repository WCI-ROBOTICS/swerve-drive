# Bill of materials
## Understanding the list
### There are two sections:
* Electronics
    * The electrical parts (Cortex, battery, motors...)
* Hardware
    * The hardware (Screws, nuts, metal...)
### Each table is split into 4 parts:
* Quantity
    * The amount of a part required
* Part name
    * The name of the part
* Part #/Kit #
    * The Vex part number where you can find the part
* Notes
    * Any notes about the part
